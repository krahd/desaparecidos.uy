## Read status

I read `doc/desaparecidos-uy-project-description.md`. The paper should not be framed as a generic AI-art paper. Its strongest contribution is:

> a computational memorial system that reconstructs public images of Uruguay’s disappeared from fragments of the present social/material country, while refusing forensic reconstruction, biometric identification, resurrection media, and technological repair.

The project document already gives the main argument: the work is a triptych — **Todos somos familiares**, **Están en todas partes**, **Seguimos buscando** — and it explicitly defines the project as neither archive, forensic system, biometric system, deepfake/resurrection media, nor technological monument.

The most publishable first version is probably based on **Están en todas partes**, because the document itself identifies it as the strongest first prototype and the lower-privacy-risk path.
